# LeadMatch — Ingeniero Binario

Plugin del curso **"Construye desde 0 tu Negocio Digital"** de LeadMatch.

## Que hace este plugin

Conecta con el servidor MCP del curso para:
- Verificar acceso del alumno (codigo de clase + email)
- Ejecutar los 6 modulos del curso con cuestionarios adaptativos
- Generar documentos estrategicos aplicados al caso real del alumno
- Guardar todo en Notion del alumno

## Skills incluidos

| Skill | Descripcion |
|-------|-------------|
| `ejecutar-modulo` | Ejecuta un modulo completo del curso |
| `progreso-curso` | Muestra el avance del alumno |
| `setup-leadmatch` | Configura el entorno inicial del alumno |

## Servidor MCP

El plugin se conecta automaticamente al servidor del curso en:
`https://leadmatch-mcp.ingenierobinario.com/mcp`

## Como empezar

1. Instala el plugin en Cowork
2. Escribe "empezar el curso" o "setup inicial"
3. Proporciona tu codigo de modulo y email cuando se te pida
4. Sigue las instrucciones del asistente

## Requisitos

- Tener acceso al curso (email registrado en Systeme.io con tag activo)
- Tener Notion conectado en Cowork
- Codigo del modulo proporcionado por el profesor en clase
